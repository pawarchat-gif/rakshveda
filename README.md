# 📌 Overview

ANTOIDX is an advanced AI-powered cyber defense and malware analysis platform developed using **Python**, **Flask**, **HTML**, **CSS**, and **JavaScript**.

The project simulates a futuristic military-grade cyber defense radar system capable of:

- Detecting malware patterns
- Analyzing hidden payloads
- Identifying suspicious files
- Performing DNA threat profiling
- Simulating real-time cyber attacks
- Visualizing threats through animated radar dashboards

The system combines AI-inspired detection logic with a visually interactive frontend to create a next-generation cybersecurity simulation platform.
